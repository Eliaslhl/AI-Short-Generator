## 🔄 HOW IT WORKS

```
USER FLOW:
/generate (Choose Source)
    ↓
/generate/youtube OR /generate/twitch
    ↓
GeneratorForm (Shared Component)
    ↓
POST /api/generate
    ↓
BACKEND:
- Source Detection (URL contains "twitch.tv"?)
- Route to Service (download_youtube or download_twitch)
- Process Pipeline (transcribe → analyze → render)
- Return Clips
    ↓
DISPLAY:
- Clip Grid
- Video Modal
- Download Options
```
