🔄 INTEGRATION WITH PREVIOUS PHASES

Phase 2b (Audio/Motion) → Phase 2c Workflow:

1. Input: Twitch VOD URL
   ↓
2. TwitchClient: Parse & validate
   ↓
3. VideoDownloadManager: Download (3-8 min/hour)
   ↓
4. _segment_video(): Chunk into 30-min segments
   ↓
5. Process Each Chunk (Parallel):
   ├─ RealAudioProcessor (librosa)
   │  ├─ RMS energy
   │  ├─ Spikes
   │  ├─ MFCC
   │  └─ Spectral features
   ├─ MotionProcessor (OpenCV)
   │  ├─ Frame diffs
   │  ├─ Scene changes
   │  └─ Optical flow
   └─ HighlightDetector
      └─ Combined scores
   ↓
6. Return highlights + metadata
